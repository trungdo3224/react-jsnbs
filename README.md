# react-jsnbs is a markdowns editor and code editor for javascript on you local browser.
# How to use react-jsnbs
# Run `npx react-jsnbs serve` and open application on localhost:4005, you can find react-jsnbs package on (https://www.npmjs.com/package/react-jsnbs)

# Or clone this repository, then:
### `lerna bootstrap` at project's root folder.
### `npm run start` at project's root folder.
### `cd packages/cli/dist` and run `node index.js serve`
